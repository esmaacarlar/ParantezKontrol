# Parantez Kontrol

Bu proje, verilen bir metin içerisindeki parantezlerin ((,{,[) doğru bir şekilde açılıp kapatıldığını kontrol eden bir Java uygulamasıdır.
• Dengeli parantezlerin kontrolü için stack mimarisi kullanılır
• Yuvarlak, süslü ve köşeli parantezlerin hiyerarşisini test eder
• Hatalı kapatma veya eksik parantez durumlarında kullanıcıya uyarı verir
